# drfRbac前端 (Vue 3 + TypeScript + Vite)

此目录包含 drfRbac 项目的前端代码。负责提供应用程序的用户界面和交互体验。

### 目录结构

frontend 目录的结构如下：

```
frontend/
  ├── src/              # 主要源代码目录
  ├── public/           # 静态资源文件
  ├── assets/           # 样式表和其他项目资源
  ├── build/            # 构建输出目录
  ├── node_modules/     # 第三方依赖项
  ├── package.json      # 项目配置文件
  └── README.md         # README 文件（您在这里）
  
```

### 项目配置

要设置和运行前端开发环境，请按照以下步骤进行操作：

1.确保您的系统上安装了 Node.js 和 npm（Node 包管理器）。

2.打开终端并导航到 frontend 目录。

3.运行 npm install 来安装项目的依赖项。

4.如有需要，请在 package.json 文件中自定义配置和设置。

5.运行 npm run dev 启动开发服务器。

6.打开浏览器并访问指定的 URL，以查看运行中的应用程序。
